Projet Pharaon Zombie

(en cours)

Logiciel utilisé : Unity
Language utilisé : C#

Le projet Pharaon Zombie est un jeu d'action-survie dans lequel on peut alterner entre la première et la troisième personne. Le jeu se déroule dans un univers égyptien, et le but est de survivre à une invasion de zombies. Pour cela, le joueur devra utiliser les armes à disposition sur la carte, qu'il pourra acheter avec de l'argent obtenu en tuant des zombies. Plus le joueur survit longtemps, plus les ennemis deviennent puissants et coriaces. Le projet est actuellement en cours de développement, et l'environnement dans lequel le joueur évoluera est également en cours de création. Une démonstration vidéo est disponible ci-dessous.
