Dracula Castle

Logiciel utilisé : Unity
Language utilisé : C#

Dracula Castle est un jeu d'action aventure à la première personne se déroulant dans un univers vampirique. Ce jeu est un projet personnel, à travers son développement, j’ai pu appliquer les connaissances que j'ai cumulées sur Unity. Je travaille toujours sur ce projet, mon but, à terme, est de créer un jeu intégral. Une démo est disponible ici.
